# Ravindra IAS Gwalior Website

A premium, cinematic, and high-conversion landing page for **Ravindra IAS Gwalior Branch**. Built with a focus on immersive storytelling, local credibility, and technical excellence.

## 🚀 Features

- **Cinematic Hero Experience**: Custom 3D-integrated hero section with a high-performance **Canvas Particle Engine**.
- **Smooth Interaction**: Integrated **GSAP** and **Lenis** for buttery-smooth scroll-based animations.
- **Leadership Showcase**: Premium glassmorphic cards for branch directors with interactive hover effects.
- **Localized Content**: Specific focus on Gwalior landmarks, testimonials, and contact details.
- **Performance Optimized**: Lazy-loaded assets and optimized canvas density for mobile compatibility.
- **Glassmorphism UI**: Modern, futuristic design with soft glows and blurred layers.

## 🛠️ Tech Stack

- **Frontend**: React (Vite)
- **Styling**: Tailwind CSS
- **Animations**: GSAP, Framer Motion
- **Scrolling**: Lenis (Smooth Scroll)
- **3D/Interactive**: Canvas API (Particles)
- **Icons**: Lucide React

## 📂 Project Structure

- `/src/components`: Reusable UI components (Navbar, Footer, Particles).
- `/src/sections`: Main page sections (Hero, About, Stats, Contact, etc.).
- `/src/assets`: Project images, logos, and portraits.
- `/src/styles`: Tailwind and global CSS configurations.

## 🏁 Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/ShivamPatidar03/Ravindra_IAS.git
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run development server**:
    ```bash
    npm run dev
    ```
4.  **Build for production**:
    ```bash
    npm run build
    ```
